# Test Práctico - Martín Luz 
### Pre-requisitos

* Nodejs
* npm

### Quick start

* Instalar las dependencias del proyecto:

  `` 
  npm install
  ``
* Levantar el servidor:

   `` 
  npm run start
  ``

* Abrir el navegador e ingresar a localhost en el puero inducado

  `` 
 http://localhost:3000
  ``
